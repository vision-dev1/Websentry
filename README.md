# WEBSENTRY

**Automated Web Vulnerability Scanner**

Websentry orchestrates automated reconnaissance and vulnerability scanning against web applications, identifying misconfigurations and standard OWASP risks.

### Core Capabilities
- **Target Profiling**: Resolves headers, TLS misconfigurations, and exposed administrative panels.
- **Payload Fuzzing**: Tests inputs for reflection, SQLi, and XSS vulnerabilities.
- **Reporting**: Exports actionable security reports for remediation.

### Usage
```bash
git clone https://github.com/Diode11-Alt/Websentry.git
cd Websentry
python3 websentry.py --target <URL>
```
